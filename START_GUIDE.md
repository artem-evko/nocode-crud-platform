# Как запустить проект Nocode CRUD Platform

Проект состоит из **бэкенда** (Java, Spring Boot) и **фронтенда** (React, Vite, Tailwind CSS). Обе части нужно запустить для полноценной работы приложения.

---

## 1. Запуск Backend (Сервер)

Убедитесь, что у вас запущена база данных PostgreSQL (используется Liquibase для накатывания миграций).

**Windows (PowerShell):**
В корневой папке проекта есть специальный скрипт запуска `start-platform.ps1`. Он автоматически проверяет, свободен ли порт 8082, и стартует сервер.

Откройте терминал PowerShell в корне проекта и выполните:
```powershell
powershell -ExecutionPolicy Bypass -File .\start-platform.ps1
```

*(Бэкенд будет доступен по адресу `http://localhost:8082`)*

---

## 2. Запуск Frontend (Пользовательский интерфейс)

Фронтенд находится в подпапке `platform-frontend`. Для его работы потребуется Node.js.

Откройте новый терминал и выполните:
```bash
# 1. Перейдите в папку с фронтендом
cd platform-frontend

# 2. Установите зависимости (один раз)
npm install

# 3. Запустите dev-сервер
npm run dev
```

*(Фронтенд будет доступен по адресу `http://localhost:5173`)*

---

## 3. Проверка работы

1. Откройте в браузере `http://localhost:5173/login`.
2. Введите учетные данные:
   - Логин: **admin**
   - Пароль: **admin**
3. Нажмите "Sign in". Вы должны попасть в дашборд проектов.
